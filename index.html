Secure, Browser-Based Data Masking & Multi-Format Exporter

When working with customer analytics, AI development pipelines, or software testing environments, handling Personally Identifiable Information (PII) safely is critical. This open-source, client-side application helps developers, compliance officers, and data scientists safely strip away sensitive fields before sharing reports or loading logs into storage formats.

Supported Data Manipulation Strategies

  • Redact (Data Masking): Replaces cell text entirely with block masking. Emails retain their domain (e.g., ████@domain.com) so downstream formats still validate.
  • Anonymize (Pseudo-anonymization): Replaces raw inputs with matching pseudo-identities (like user001@example.com or Value_001). Identical raw values map consistently across rows.
  • Shuffle (Row Mixing): Breaks horizontal correlations across sensitive data by shuffling row order within a single column.
  • Keep As-Is: Preserves the column exactly as uploaded.

Frequently Asked Questions (FAQ)

Is it secure to upload corporate files to this tool?

This application is static and operates entirely within client-side memory using your browser's local sandbox. Your file is parsed directly in your device's memory using HTML5 file handles and is never transmitted to a remote server.

Delete this column?

Remove and its data from every row. You can't undo this.
